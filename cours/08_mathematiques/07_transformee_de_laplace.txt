CHAPITRE 7 — TRANSFORMÉE DE LAPLACE
BTS Génie Logiciel — Mathématiques

1. DÉFINITION ET MOTIVATION

La transformée de Laplace convertit une fonction du temps f(t) en une fonction F(s) de la variable complexe s = σ + jω. Elle simplifie la résolution des équations différentielles (les dérivées deviennent des multiplications).

Transformée de Laplace bilatérale :
F(s) = L{f(t)} = ∫[-∞, +∞] f(t) × e^(-st) dt

Transformée unilatérale (la plus utilisée en automatique et traitement du signal) :
F(s) = L{f(t)} = ∫[0, +∞] f(t) × e^(-st) dt

Existence : F(s) existe si f est d'ordre exponentiel, c'est-à-dire |f(t)| ≤ M×e^(αt) pour t ≥ 0.
La transformée existe pour Re(s) > α (demi-plan de convergence).

2. TABLE DES TRANSFORMÉES FONDAMENTALES

f(t) (t ≥ 0)                  F(s) = L{f(t)}
-----------------------------  ---------------------
δ(t) (impulsion de Dirac)      1
u(t) (échelon unité)           1/s
t (rampe)                      1/s²
tⁿ                             n! / s^(n+1)
e^(at)                         1 / (s − a)
sin(ωt)                        ω / (s² + ω²)
cos(ωt)                        s / (s² + ω²)
e^(at) × sin(ωt)               ω / [(s−a)² + ω²]
e^(at) × cos(ωt)               (s−a) / [(s−a)² + ω²]
e^(at) × tⁿ                    n! / (s−a)^(n+1)

3. PROPRIÉTÉS FONDAMENTALES

Linéarité :
L{a×f(t) + b×g(t)} = a×F(s) + b×G(s)

Décalage en fréquence :
L{e^(at) × f(t)} = F(s − a)

Décalage temporel (retard) :
L{f(t − t0) × u(t − t0)} = e^(-t0×s) × F(s)

Dérivation dans le domaine temporel :
L{f'(t)} = s × F(s) − f(0⁺)
L{f''(t)} = s² × F(s) − s×f(0⁺) − f'(0⁺)
L{f^(n)(t)} = s^n × F(s) − s^(n-1)×f(0⁺) − ... − f^(n-1)(0⁺)

Intégration :
L{∫[0,t] f(τ)dτ} = F(s)/s

Théorème de la valeur initiale :
lim(t→0⁺) f(t) = lim(s→∞) s × F(s)

Théorème de la valeur finale :
lim(t→∞) f(t) = lim(s→0) s × F(s)   (si les pôles de s×F(s) sont à partie réelle strictement négative)

4. TRANSFORMÉE INVERSE

Transformée de Laplace inverse : L⁻¹{F(s)} = f(t)
Méthode : décomposer F(s) en éléments simples, puis utiliser la table.

Décomposition en éléments simples :
Pôles simples réels : A/(s − a) → A × e^(at)
Pôles complexes conjugués : (as + b)/[(s+α)² + ω²] → utiliser les formes sin/cos

Exemple :
F(s) = (2s + 1) / (s² + 3s + 2) = (2s + 1) / [(s+1)(s+2)]

Décomposition : A/(s+1) + B/(s+2)
A = [(2s+1)/(s+2)]|s=-1 = (−2+1)/(−1+2) = −1/1 = −1
B = [(2s+1)/(s+1)]|s=-2 = (−4+1)/(−2+1) = −3/(−1) = 3

F(s) = −1/(s+1) + 3/(s+2)
f(t) = −e^(−t) + 3×e^(−2t)

5. APPLICATION : RÉSOLUTION D'ÉQUATION DIFFÉRENTIELLE

Résoudre : y'' + 3y' + 2y = e^(−t), y(0) = 0, y'(0) = 0

Prendre la transformée de Laplace :
s²Y(s) + 3sY(s) + 2Y(s) = 1/(s+1)
Y(s) × (s² + 3s + 2) = 1/(s+1)
Y(s) = 1/[(s+1)(s+1)(s+2)] = 1/[(s+1)²(s+2)]

Décomposition : A/(s+1) + B/(s+1)² + C/(s+2)
C = [1/(s+1)²]|s=-2 = 1/(−1)² = 1
B = [1/(s+2)]|s=-1 = 1/(1) = 1
A = d/ds[1/(s+2)]|s=-1 = −1/(s+2)²|s=-1 = −1

Y(s) = −1/(s+1) + 1/(s+1)² + 1/(s+2)
y(t) = −e^(−t) + t×e^(−t) + e^(−2t) = e^(−t)(t − 1) + e^(−2t)
